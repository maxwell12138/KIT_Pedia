## 参与方式
* 每位参与者根据框架内的内容，记录自己对某话题的认识与了解。<br>
* 内容不仅包括客观描述，更提倡记录主观感受。比如某门课特别难，某个地方特别好玩，英国签证要多久才能办下来，苏黎世的披萨特别贵，等等等等。<br>
* 内容禁止涉及违法内容，禁止人身攻击，禁止广告。<br>
* 每一个知识节点以word的形式，提交到相应的目录下。word中开头应该包括作者的姓名（化名亦可），以及最新更新时间。未留下姓名默认同意佚名发表。如果愿意，也可以留下自己的联系方式，方便后人咨询相关问题。<br>
* 对于已存在的节点，如果不同意其中观点，或者时移世易情况改变，也可提交自己的了解。<br>
* 不擅长使用Github的参与者，可以将文章通过电子邮件发送给管理员（fulinou@foxmail.com）。管理员将代为上传。<br>
* 每一个完成的节点，将在构架图中被点亮。<br>
* 所有内容将被汇总，以PDF形式，供学生免费下载阅读。本项目承诺不用于商业用途，也禁止他人或机构用于商业用途。本项目保留追究其侵权行为的权利。<br>
